Welcome to Rudix!

Rudix is a build system target on macOS and offers pre-built packages for this platform. The build system has some support for Linux, OpenBSD, FreeBSD and NetBSD. If you want to know more about the project and want to contribute, please, visit us on github.com/rudix-mac.

Packages for macOS

These packages are known to work on El Capitan (OS X 10.11), Mojave (macOS 10.14) and Catalina (macOS 10.15). Check the MD5, SHA-1, SHA-256 and the file SIZES.
